# X-PPO

## Installation

*Requirement: python 3.7+*
* clone this repo
* cd to the repo
* install requirement: `pip install -r requirements.txt`
* run train: `python3 framework/train.py`
* run test: `python3 framework/test.py`

**Tested on macOS Catalina**
